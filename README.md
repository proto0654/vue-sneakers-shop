# Vue Sneakers Shop

## Демо

[Открыть на GitHub Pages](https://proto0654.github.io/vue-sneakers-shop/)

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на GitHub Pages

```bash
npm run deploy
```

- После деплоя проект будет доступен по ссылке выше.
- Для корректной работы роутинга используется hash-режим.
